TMI.php

The Twitch Message Interface for PHP 7.4+

Inspired by tmi.js, php-irc-client this package is a full featured, high performance Twitch IRC client written in PHP 7.4.

Also have a look at ghostzero/tmi-cluster. TMI Cluster is a Laravel package that makes the PHP TMI client scalable.
